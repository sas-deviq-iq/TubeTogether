# 🎬 TubeTogether - Premium Media Streaming App

**TubeTogether** هو تطبيق أندرويد متكامل وعالي الأداء لمشاهدة الأفلام والمسلسلات، تم تصميمه وبناؤه باستخدام أحدث تقنيات تطوير الأندرويد (Modern Android Development). يقدم التطبيق تجربة مستخدم (UX) سينمائية وتصميم واجهة (UI) سلس وعصري مستوحى من كبرى منصات البث العالمية.

## ✨ المميزات الرئيسية (Key Features)

* **تجربة تشغيل سينمائية (Immersive Player):** مشغل فيديو متطور مبني على مكتبة `Media3 ExoPlayer`، يعمل بالوضع العرضي التلقائي (Landscape) بكامل الشاشة مع إخفاء ذكي لشريط الإشعارات والتنقل.
* **إعدادات تحكم ذكية (Smart Controls):** قائمة إعدادات سفلية (Modal Bottom Sheet) متطورة تتيح للمستخدم:
  - التبديل بين دقات الفيديو المختلفة (جودات متعددة) بسلاسة.
  - التحكم بسرعة التشغيل (x0.5 إلى x2.0).
  - تخصيص أزرار تخطي الفيديو (5 أو 10 أو 15 ثانية).
* **نظام التحديثات المدمج (In-App Updater):** نظام احترافي يقوم بالبحث عن التحديثات من خوادم `GitHub Releases`، وتحميل الـ `APK` في الخلفية باستخدام `DownloadManager`، وفتح شاشة التثبيت تلقائياً عند توفر إصدار أحدث.
* **تصفح وبحث متقدم:** واجهات ديناميكية لاستكشاف أحدث الأفلام، بالإضافة إلى محرك بحث سريع لعناوين الأفلام مع تفاصيل دقيقة (القصة، طاقم العمل، التقييم، الإعلان التشويقي).
* **التخزين المحلي وحفظ التقدم (Local Storage):** نظام لحفظ الأفلام المفضلة وحفظ "نقطة توقف المشاهدة" ليعود المستخدم لاستكمال الفلم من حيث توقف تماماً.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **اللغة الأساسية:** Kotlin 100%
* **واجهة المستخدم (UI):** Jetpack Compose (Material Design 3)
* **مشغل الميديا:** AndroidX Media3 (ExoPlayer)
* **الشبكات وجلب البيانات:** Retrofit2 & OkHttp3
* **تحميل وعرض الصور:** Coil (مُحسّن لجلب الصور من خوادم الـ CDN بسرعة)
* **الملاحة (Navigation):** Jetpack Navigation Compose
* **البنية التحتية للخوادم (Backend Proxy):** تم دمج التطبيق مع نظام `Node.js Proxy` (مُستضاف على VPS) لتجاوز قيود الخوادم (Geo-Blocking) ومعالجة الـ Headers للوصول إلى سيرفرات البث بجودة عالية.

## 📱 نظرة على المعمارية (Architecture Highlights)

- **Jetpack Compose First:** تم بناء التطبيق بالكامل باستخدام Compose بدون استخدام الـ XML (باستثناء الإعدادات الضرورية)، مما يوفر أداءً استثنائياً في الـ Rendering.
- **Asynchronous Flow:** معالجة البيانات والشبكات عبر الـ `Coroutines` لضمان عدم توقف واجهة المستخدم مطلقاً.
- **FileProvider Integration:** استخدام صلاحيات النظام العميقة لتحميل وتثبيت ملفات الـ APK الخاصة بالتحديثات بشكل آمن تماماً عبر الـ `Android Package Installer`.
